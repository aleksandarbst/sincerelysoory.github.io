Process

I've recently started falling in love with apple's ui style patterns, and there is a definitely a note of inspiration in the way items are diplayed. Though, I didn't use the stereotipically easy to use San Francisco font (rest in peace Helvetica), and decided to go with Inter, a great all-rounder font (which is also plastered all over this site). This redesign was really built in mind with the larger size of the iPhone X and it's nearly non-existant bezels.
